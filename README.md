# PeerSync-Plateforme-de-Tutorat-ENAA

#Réaliser par : Mouhamadou FADAL Dramé

 ____________Description______________________

PeerSync est une plateforme de tutorat développée en PHP orienté objet permettant :

aux apprenants de créer des demandes d’aide,
aux tuteurs d’accepter et résoudre des demandes,
aux apprenants de laisser des avis,
à l’administrateur de superviser la plateforme via un dashboard.


_______________Téchnologie utilisées______________________

PHP 8
MySQL
PDO
TailwindCSS
Jira
ERD
Star UML
Git Hub
Workbranch
XAMPP


____________________Fonctionnalités_______________________________



   1-Authentification
Connexion utilisateur
Déconnexion
Gestion des sessions
Gestion des rôles :
admin
tuteur
apprenant


    2-Gestion des demandes

Créer une demande d’aide
Afficher les demandes
Accepter une demande
Résoudre une demande

    3-Gestion des compétences

Points forts
Ajouter des compétences fortes
Points faibles
Ajouter des compétences faibles

    4-Compatibilité

Le système détecte automatiquement les compatibilités entre :

les compétences faibles des apprenants
les compétences fortes des tuteurs


    5- Avis

Les apprenants peuvent :

noter un tuteur
laisser un commentaire après résolution d’une demande


    6- Dashboard Admin

L’administrateur peut voir :

nombre total de demandes
demandes résolues
nombre de tuteurs
nombre d’apprenants
top tuteurs
compatibilités
avis des apprenants
